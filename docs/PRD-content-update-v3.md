## 3.6 Calendly Integration (NEW)

**Type:** Floating popup widget
**Placement:** Site-wide (accessible from any page)

#### Implementation Details
```
Widget Type: Calendly Popup Widget
Trigger: Floating button (bottom-right corner)
Button Text: "Book a Consultation" or "Schedule a Meeting"
Behavior: Opens Calendly scheduling modal on click
```

#### Technical Requirements
- Add Calendly widget script to `_layouts/default.html`
- Configure with brand colors to match site design
- Widget appears on all pages
- Mobile responsive

#### Calendly Widget Code (Template)
```html
<!-- Calendly badge widget begin -->
<link href="https://assets.calendly.com/assets/external/widget.css" rel="stylesheet">
<script src="https://assets.calendly.com/assets/external/widget.js" type="text/javascript" async></script>
<script type="text/javascript">
  window.onload = function() {
    Calendly.initBadgeWidget({
      url: 'https://calendly.com/CALENDLY_USERNAME',
      text: 'Book a Consultation',
      color: '#5d7187',
      textColor: '#ffffff'
    });
  }
</script>
<!-- Calendly badge widget end -->
```

#### Client Action Required
- ⚠️ **BLOCKED**: Create Calendly account
- Provide Calendly URL (e.g., calendly.com/intentional-group)
- Set up event types (e.g., "30-min Consultation", "Project Discussion")
