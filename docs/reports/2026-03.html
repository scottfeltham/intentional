Security advisories resolved

Known vulnerabilities that were patched by the updates above.

  • HIGH Astro — authentication bypass via double URL encoding
  • HIGH Astro — reflected XSS via server islands feature
  • HIGH Astro — arbitrary local file read in dev server
  • HIGH Astro — middleware bypass via URL-encoded values
  • HIGH Astro — X-Forwarded-Host reflected without validation
  • MODERATE devalue — prototype pollution and sparse array amplification
  • MODERATE esbuild — dev server responds to any website's requests
  • MODERATE vite — multiple moderate advisories (transitive)

Code migrated for the new version

Small internal adjustments required by the framework upgrade.

  • Content collections moved from legacy API to the new loader-based API (required by Astro 6)
  • Page transition component renamed from <ViewTransitions /> to <ClientRouter />
  • Content config relocated from src/content/config.ts to src/content.config.ts
